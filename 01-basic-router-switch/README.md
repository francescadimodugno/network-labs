# Basic Router and Switch Configuration

## Scenario

A small network composed of:

- 1 Router
- 1 Switch
- 2 PCs

All devices belong to the same network segment.

## Technologies

- Cisco IOS CLI
- Basic Router Configuration
- Basic Switch Configuration
- Switch Management Interface (SVI)
- IPv4 Addressing
- Default Gateway Configuration

## Objectives

- Configure hostnames on the router and switch
- Configure privileged EXEC passwords
- Configure a MOTD banner
- Configure the router interface connected to the switch
- Configure a management IP address on the switch
- Assign IP addresses to end devices
- Configure the correct default gateway
- Verify end-to-end connectivity
- Save the device configurations

## Validation

- PCs can communicate with each other
- PCs can reach the router
- Switch management IP is reachable
- Router and switch configurations are saved successfully
