# OSPF Multi-Site Network with WAN Redundancy
## Scenario

A company with four locations:
- Rome (HQ)
- Milan
- Turin
- Datacenter

## Technologies
- VLANs
- Router-on-a-Stick
- OSPF
- DHCP Relay
- DNS
- Web Server

## Objectives
- Dynamic routing between all sites
- Centralized DHCP service
- WAN redundancy
- OSPF failover testing

## Validation
- Verified OSPF neighbor relationships
- Tested DHCP assignment across remote sites
- Confirmed network convergence after WAN link failure
