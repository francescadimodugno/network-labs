# network-labs
This repository contains networking projects and labs created with Cisco Packet Tracer.

# Topics Covered
- VLAN segmentation
- Trunking
- Router-on-a-Stick
- Static Routing
- OSPF Dynamic Routing
- DHCP and DHCP relay
- ACLs
- NAT/PAT
- Multi-Site Network Design
- WAN Redundancy Testing and Failover

These labs are part of my transition into IT networking and infrastructure. More projects will be added as I continue learning networking and Cisco technologies.
